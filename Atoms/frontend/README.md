# Frontend

React + Vite operator interface for the traffic digital twin.

Stack:

- React
- React Router
- Vite
- Tailwind CSS
- `@react-three/fiber`
- `@react-three/drei`

Views:

- Home
- Simulation
- Dashboard
- Impact

Run:

```powershell
cd frontend
npm install
npm run dev
```

The client connects to:

- `ws://localhost:8000/ws`

Build:

```powershell
cd frontend
npm run build
```
